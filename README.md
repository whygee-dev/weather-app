# Weather App 

## Introduction
Weather App is a PWA prototype application for the devchallenge https://devchallenges.io/challenges/mM1UIenRhK808W8qmLWv.

Made with NextJS, TypeScript.

## Installation

```
npm install
```

or

```
yarn install 
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```
